/ Памятка использования Git

Простые действия

Связка локального репозитория с удалённым

git remote add origin "url"

Клонировать удалённый репозиторий

git clone "url"

Добавить файлы в коммит

git add .

Создать коммит с названием

git commit -m "Добавил новую функцию"

Отправить коммит на сервер

git push

Удалить последние коммиты

git reset --hard HEAD~n (Где n — количество последних коммитов)

Добавить изменения в существующий коммит

git commit --amend

Ветки git

Создание другой ветки

git checkout -b "branch-name"

Переключение на ветку

git checkout "branch-name"

Слияние веток

git merge --no-ff feature/add-login "branch-name" (Создаёт отдельный merge-коммит. Сохраняет историю ветки.)
